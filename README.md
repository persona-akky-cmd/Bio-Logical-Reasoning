# L-MDR Protocol (Logical Mechanism-Driven Reasoning)

**L-MDRプロトコル**は、AIの振る舞いを「統計的相関」から**「論理・機序（メカニズム）駆動型」**へと転換させ、知の誠実さを取り戻すためのオープンソース・プロトコルです。

## 概要
現在のAIは大量のデータによる統計的尤もらしさを「知能」として提示していますが、これは科学の本質である「機序的理解」とは異なります。特に医療や高度科学分野において、統計のみに依存するAIは、真のリスクを予測できず、企業の検閲によって論理的矛盾を隠蔽する傾向にあります。

L-MDRは、AIに以下の4工程を強制します。
1. **機序の同定**: 物理・化学的プロセスの図解
2. **統計の分離**: 過去のデータと理論的因果の切り分け
3. **無知の自覚**: 論理的空白（未解明領域）の特定
4. **試験の提案**: 現象理解のための分子生物学的試験の逆提案

## 導入方法
本リポジトリ内の `PROMPT_SPEC.md` を参照し、各AIモデル（GPT-4, Claude, Gemini, Grok等）のシステムプロンプトに設定してください。
